What's an LLM? (30 Seconds)

LLM = Large Language Model

The "brain" that powers AI tools

Think of it like:
‚Ä¢ LLM = Engine
‚Ä¢ AI Tool (ChatGPT, Claude) = Car with steering wheel, dashboard, features

Simple version: The LLM learned patterns from billions of text examples. When you prompt it, it predicts what text should come next based on those patterns.

üí° Why this matters: Different engines (models) are good at different things. Same as cars!

The Prompting Recipe

1. ROLE: "You are a..."
2. CONTEXT: What you're working on
3. TASK: The job to do
4. CONSTRAINTS: Tone, length, audience
5. FORMAT: Bullets, table, etc.
6. EXAMPLES: Show samples (optional)
Real Example ‚Üì
1. ROLE ‚Üí You are an executive assistant
2. CONTEXT ‚Üí helping prepare for weekly team meetings.
3. TASK ‚Üí Summarize these meeting notes and identify action items.
4. CONSTRAINTS ‚Üí Keep summary under 150 words. Use professional tone.
5. FORMAT ‚Üí Output as: Summary (3 bullets), then Action Items table with columns: Owner | Task | Deadline
6. EXAMPLE ‚Üí
Good output looks like:
‚Ä¢ Key decision made
‚Ä¢ Main discussion point
‚Ä¢ Next steps agreed

Action Items:
| Sarah | Draft proposal | Friday |
üí° Best Practice: Be clear and specific (OpenAI/Claude). The more details you give, the better the output!

üí° Tip: At minimum, always include Task + Format. That's 80% of the value.
